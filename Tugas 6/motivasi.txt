Motivasi saya mengikuti bootcamp Laravel Web Dev adalah untuk memperdalam pemahaman mengenai framework Laravel yang banyak digunakan dalam pengembangan aplikasi web modern. Dengan mengikuti bootcamp ini, saya berharap dapat meningkatkan keterampilan teknis, memahami best practice dalam pengembangan aplikasi, serta mampu membangun project nyata yang dapat menambah portofolio saya. Harapan saya setelah bootcamp adalah bisa lebih percaya diri untuk mengembangkan aplikasi web secara profesional dan membuka peluang karier yang lebih baik di bidang IT.
